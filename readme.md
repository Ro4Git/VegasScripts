Scripts description: 

SnapAllEvents.cs
Take a selection of events and snap them to each other in the order of the selection
(Exist with slight overlap variations)

ResizeAllEvents:
Same as SnapAllEvents but also resize them to 1s, 0.1s or 0.01s
